BN随机密码生成器 v1.0
====================

【功能说明】
- 生成以BN开头的16位复杂密码
- 支持批量生成（1-1000个）
- 一键保存到文件
- 自动复制文件路径到剪贴板

【使用方法】
1. 输入要生成的密码数量（1-1000）
2. 点击"生成密码"按钮
3. 点击"保存密码"将结果保存到文件
4. 保存后文件路径会自动复制到剪贴板
5. 按 Win+R，粘贴路径，回车即可访问文件

【密码格式】
- 长度：16位
- 格式：BN + 14位随机字符
- 字符集：大小写字母 + 数字 + 特殊符号

【文件保存位置】
程序所在目录下的 BN_Passwords_时间戳.txt 文件

【系统要求】
- Windows 7/8/10/11
- .NET Framework 4.8 或更高版本

【注意事项】
- 请妥善保管生成的密码
- 建议定期更换重要密码
- 密码文件包含明文，请注意安全
